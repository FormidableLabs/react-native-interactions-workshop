## Getting started

### Global dependencies

```sh
yarn add --global expo-cli
```

If using your phone, install the [Expo App](https://expo.io/tools#client).

If using simulator, we recommend using the iOS simulator for this workshop.

### Start workshop application

```sh
cd app
yarn
yarn start
```
